Flight Status Checker

Constructed a replica of the FlightAware website in which you can track flights and airport boards in a portable method using Python. A variety of options (such as flight departure / arrival times, radar boards, airline information, etc.) for the user were provided in exchange for not using real time data using manually imported .JSON files from the FlightAware API.
